TexText for Inkscape 1.0 beta on Windows¶

This guide explains how to install and use TexText for Inkscape 1.0 beta together with Inkscape 1.0 beta on a system on which also Inkscape 0.92.x is installed.

Note

Several of the following actions take place in a command window. To open a command window from the Windows Explorer right click while pressing the SHIFT key in the directory you want to open it. Then select Open command window here.

Preparation¶

In a command window execute the following steps:

  1. Backup your existing Inkscape extension directory:

    xcopy /S /E /Y %APPDATA%\inkscape\extensions %APPDATA%\inkscape\extensions.backup\

  2. Remove the old TexText installation:

    del /S /Q %APPDATA%\inkscape\extensions\textext
del /Q %APPDATA%\inkscape\extensions\textext.inx

Download and install the Inkscape 1.0 beta app image file¶

  1. Download Inkscape 1.0 beta from https://inkscape.org/release/1.0beta2/.
  2. Extract the zip package to a folder of your choice, e.g. C:\InkscapeBeta.
  3. Veryify that you are able to launch Inkscape 1.0 beta by double clicking on inkscape.exe in C:\InkscapeBeta\bin.

Download and install TexText for Inkscape 1.0 beta¶

  1. Download the most recent preview package from GitHub release page

  2. Extract the package and change into the created directory.

  3. In a command window run setup_win.bat with (Important!!) specification of the path to your Inkscape 1.0 beta executable from your terminal:

    setup_win.bat /d:"C:\InkscapeBeta\bin"

    Setup will inform you if some of the prerequisites needed by TexText for Inkscape 1.0 beta are missing. Install them.

    Important

    1. Compared to previous versions of TexText for Inkscape 0.92.x TexText for Inkscape 1.0 beta does not need any conversion utilities like ghostscript, pstoedit or pdfsvg. Furthermore, the required Python bindungs for the GTK-GUI are already included in the windows version of Inkscape 1.0 beta.
    2. Currently, GTKSourceView is not available in the Windows version of TexText for Inkscape 1.0 beta, hence syntax highlighting is not enabled. An installer will be provided soon.

Now you can launch Inkscape 1.0 beta by double clicking on inkscape.exe in C:\InkscapeBeta and work with TexText for Inkscape 1.0 beta

Please report any issues! Thank you!

Switching back to Inkscape Inkscape 0.92.x¶

In a command window execute:

move /Y %APPDATA%\inkscape\extensions %APPDATA%\inkscape\extensions.beta\
xcopy /S /E /Y %APPDATA%\inkscape\extensions.backup %APPDATA%\inkscape\extensions\
